¿Qué es CSS y para qué sirve?

CSS es el maquillaje de la web. Su propósito es que el HTML se vea bonito.

Cómo aplicar CSS:

  • Inline: Directo en la etiqueta (ej. style="").
  • Interno: Dentro de <style> en el head.
  • Externo: En un archivo .css aparte (el mejor).

Selector vs Regla:

El Selector es a quién apuntas (ej. p) y la Regla es lo que le haces (ej. color: aqua).

SELECCIONANDO COSAS EN CSS

Los "Apuntadores" (Selectors)

  • Tag Selector: Selecciona a TODOS los de un tipo.
  • Class Selector: Selecciona a un grupo con una etiqueta específica. Usas un punto
  • Id Selector: Selecciona a UNO solo, es único. Usas un gato
  • Tag and Class: Selecciona un tipo específico que tenga esa etiqueta.

¿Qué es Class e ID?

Class: Es una etiqueta que puedes repetir. Muchos elementos pueden tener la misma clase.

ID: Solo puede haber uno por página. Es para cosas únicas.

Relación con CSS: Son los nombres que usamos en el archivo CSS para saber a quién cambiarle el color o tamaño.

Box Model

En HTML, todo es una caja invisible. Se compone de:

  1. Content: El texto o imagen.
  2. Padding: El espacio entre el regalo y la caja.
  3. Border: El borde de la caja.
  4. Margin: El espacio entre esta caja y la caja de al lado.
